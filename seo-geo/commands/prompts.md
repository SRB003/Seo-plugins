# Prompt Research

Discover the AI prompts and sub-queries relevant to your business.

## Usage
```
/seo-geo:prompts <topic-or-niche> [--competitor competitor-url]
```

## What it does
1. Generates prompt variations users would ask AI about your topic
2. Maps sub-query fan-out patterns for each prompt
3. Checks which prompts your content currently covers
4. Identifies prompt gaps where you have no coverage
5. Compares against competitor coverage

Uses the `semantic-footprint` skill.
