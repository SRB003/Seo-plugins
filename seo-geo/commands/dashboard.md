# GEO Dashboard

Master overview of generative engine optimization metrics.

## Usage
```
/seo-geo:dashboard <site-url>
```

## What it does
Aggregates all GEO data from Supabase:
- Share of Model trend
- Content GEO scores by page
- Semantic footprint coverage
- AI crawler access status
- Authority score
- Citation trends across platforms
- Prompt coverage gaps

Presents a unified dashboard for AI visibility management.
