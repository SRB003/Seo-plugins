# AI Crawler Check

Verify your site is accessible to AI search engine crawlers.

## Usage
```
/seo-geo:crawlers <site-url>
```

## What it does
1. Checks robots.txt for AI bot blocks (GPTBot, ChatGPT-User, PerplexityBot, ClaudeBot, Google-Extended)
2. Tests if pages are server-rendered vs client-side JS only
3. Checks for Cloudflare or CDN AI bot blocking
4. Verifies page load speed
5. Reports which AI engines CAN and CANNOT access your content

Critical first step — if AI can't read your content, no amount of optimization helps.
