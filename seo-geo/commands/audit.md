# GEO Audit

Audit content for Generative Engine Optimization readiness.

## Usage
```
/seo-geo:audit <url> [--keyword <keyword>]
```

## What it does
1. Analyzes content against all 5 GEO techniques (citations, statistics, quotes, fluency, terminology)
2. Checks AI crawler accessibility (robots.txt, bot blocking)
3. Assesses content extractability for RAG systems
4. Measures fact density and source quality
5. Scores GEO readiness with specific improvement recommendations

Uses the `geo-content-optimizer` skill.
