# AEO Optimize

Restructure content for answer engine optimization.

## Usage
```
/seo-aeo:optimize <url-or-content> --keyword <keyword> [--format paragraph|list|table]
```

## What it does
1. Analyzes the target content against AEO best practices
2. Identifies target questions from PAA and AI query patterns
3. Restructures content with answer-first paragraphs
4. Adds question-format headings matching real user prompts
5. Enhances fact density with statistics and citations
6. Generates AEO-optimized schema markup
7. Adds Speakable schema for voice assistant optimization
8. Presents before/after with AEO score improvement

Uses `answer-optimizer` and `featured-snippet-hunter` skills together.
