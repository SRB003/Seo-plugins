# Featured Snippet Opportunities

Discover and target featured snippet opportunities.

## Usage
```
/seo-aeo:snippets <site-url> [--keywords kw1,kw2,kw3]
```

## What it does
1. Checks which keywords already have featured snippets
2. Identifies snippets you own (to defend) vs don't own (to steal)
3. Finds keywords where no snippet exists (new opportunities)
4. Analyzes current snippet format (paragraph/list/table)
5. Generates optimization recommendations per keyword
6. Mines People Also Ask for additional opportunities

Uses the `featured-snippet-hunter` skill.
