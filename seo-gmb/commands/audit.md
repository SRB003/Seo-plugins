# GBP Audit

Run a complete Google Business Profile audit.

## Usage
```
/seo-gmb:audit <business-name> --city <city> [--compare-competitors]
```

## What it does
1. Searches for and analyzes the business's GBP listing
2. Scores across 6 categories: completeness, visuals, reviews, posts, NAP, website alignment
3. Benchmarks against local pack competitors
4. Identifies critical gaps and quick wins
5. Stores audit in Supabase for trend tracking

Uses the `gbp-audit` skill.
