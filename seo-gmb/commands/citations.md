# Citation Check

Audit NAP consistency across business directories.

## Usage
```
/seo-gmb:citations <business-name> --phone <phone> --address <address>
```

## What it does
1. Establishes canonical NAP from the GBP listing
2. Searches 15+ directories for the business listing
3. Compares NAP data against the canonical
4. Flags inconsistencies with specific fix instructions
5. Detects duplicate listings
6. Generates a prioritized fix plan

Uses the `citation-checker` skill.
