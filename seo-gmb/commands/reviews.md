# Review Management

Analyze reviews and draft responses.

## Usage
```
/seo-gmb:reviews <business-name> [--respond] [--analyze] [--request-templates]
```

## What it does
- `--respond`: Draft personalized responses for all unresponded reviews
- `--analyze`: Sentiment analysis and keyword extraction across all reviews
- `--request-templates`: Generate review request email/SMS templates
- All flags: Run everything

Uses the `review-manager` skill. All response drafts require approval before posting.
